## Setup

### Dependencies

- [neovim](https://github.com/neovim/neovim)
- [vim-plug](https://github.com/junegunn/vim-plug)
- [coc](https://github.com/neoclide/coc.nvim)

### Installation

1. Clone the repo into `~/.config/nvim/`
1. Run `:PlugInstall` and restart nvim
1. Run `:CocInstall`
